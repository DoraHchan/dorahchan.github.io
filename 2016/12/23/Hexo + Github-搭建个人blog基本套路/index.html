Dec 22, 2016

搭建环境

[] 安装 git ————用于同步本地以及服务器端文件，简言之，至于为什么用这么高大上的东西？都打算自己搭建blog了，版本控制器对于更新纠正数据也会便捷许多
[] 安装 nodeJs ———— Hexo 的 blog 框架是依赖nodeJs这个包的，没有 nodeJs，blog 没有框架就好像漂移的车没了车架
[x] 以上安装都是满足本地的 blog 书写环境，以及日后搭建的平台能显得有格调，内涵

服务端需求

[] github 能为个人用户提供300M的存储空间，用作个人 blog 数据管理，所以还是偏向使用 github，不过目前国内两大云厂商都有一堆福利，觉得麻烦的话，也可以使用这两家的，企鹅家的优惠还行，有各种优惠券，阿里家的满足基础消费可以免费用半年，加上消费的也基本够用了
[] 这里还是会侧重 github pages
[] GitHub Pages 本用于介绍托管在 GitHub 的项目， 不过，由于他的空间免费稳定，用来做搭建一个博客再好不过了。
[] github pages 可以在开始创建第一个项目后，setting 生成 github pages
[] 生成后，可以得到 ‘yourname.github.io’ 的网址，这边是个人博客页面
