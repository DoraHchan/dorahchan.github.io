Github 搭建个人blog基本套路

Dec 22, 2016

##搭建环境

  • 安装git ————用于同步本地以及服务器端文件，简言之，至于为什么用这么高大上的东西？都打算自己搭建blog了，版本控制器对于更新纠正数据也会便捷许多
  • 安装nodeJs ————hexo的blog框架是依赖nodeJs这个包的，没有nodeJs，blog没有框架就好像漂移的车没了车架
    以上安装都是满足本地的blog书写环境，以及日后搭建的平台能显得有格调，内涵

##服务端需求

  • github能为个人用户提供300M的存储空间，用作个人blog数据管理，所以还是偏向使用github，不过目前国内两大云厂商都有一堆福利，觉得麻烦的话，也可以使用这两家的，企鹅家的优惠还行，有各种优惠券，阿里家的满足基础消费可以免费用半年，加上消费的也基本够用了
